# Live Stock Applicaiton

Javascrit/JQuery based Live Stock Application 

## Installation

1. Clone this repo
```git clone https://github.com/tusharv/LSA```

2. Install dependencies ```npm install```

3. Run ```npm start```

4. Open ```http://127.0.0.1:8080/docs```
